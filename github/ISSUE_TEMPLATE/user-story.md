As a product owner

I need the ability to create a product in the catalog

So that new products can be added to the e-commerce platform

## Acceptance Criteria

Given the catalog administrator is logged into the system and has entered a valid product name, description, and price

When the administrator clicks the Create Product button

Then the new product is saved and displayed in the product catalog
